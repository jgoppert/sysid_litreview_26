# Output Error Method

This folder contains the classical output-error benchmark used to recreate and
extend the current paper figures.

Implemented:

- Single-shooting OEM
- Multiple-shooting OEM
- Trajectory-fit figure
- Parameter-error figure
- Trajectory-RMSE figure
- Computational-cost figure
- Local identifiability figure

Run:

```bash
python3 methods/output_error/oem_benchmark.py
```
